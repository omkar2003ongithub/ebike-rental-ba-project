## Project Overview

This project simulates a real-world BA engagement for a startup specializing in **eco-friendly urban mobility through e-Bike rentals**. The startup targets metropolitan cities, serving tourists, urban commuters, corporate clients, and gig workers.

All analysis, documentation, and deliverables follow **BABOK v3 Knowledge Areas** as the guiding framework, with Agile practices incorporated across elicitation, requirements management, and solution evaluation phases.

| Attribute        | Details                        |
|------------------|-------------------------------|
| Role             | Business Analyst               |
| Domain           | Urban Mobility / EV Rentals    |
| Framework        | BABOK v3                       |
| Methodology      | Agile Scrum                    |
| Project Type     | Case Study                     |
| Target Market    | Metropolitan Cities (India)    |
